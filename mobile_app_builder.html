1 🚀 الطريقة الأسهل - GitHub Actions

بناء تلقائي في السحابة - الأسهل والأسرع

📁 إنشاء Repository جديد
📋 خطوات سريعة:
1. أنشئ Repository جديد في GitHub
2. ارفع جميع ملفات المشروع
3. اذهب إلى Actions → Build APK
4. اضغط "Run workflow"
5. انتظر 20-30 دقيقة
6. حمل APK من Artifacts
git init
git add .
git commit -m "Mobile app ready"
git remote add origin https://github.com/YOUR_USERNAME/math-competition-mobile.git
git push -u origin main

2 🐧 البناء المحلي - Linux/WSL

للمطورين المتقدمين - تحكم كامل

⚠️ متطلبات:
• Linux أو WSL على Windows
• 8GB RAM على الأقل
• 20GB مساحة فارغة
• اتصال إنترنت سريع
# تثبيت المتطلبات
sudo apt update && sudo apt upgrade -y
sudo apt install -y openjdk-8-jdk git zip unzip autoconf libtool
pip3 install buildozer cython kivy kivymd

# بناء APK
buildozer android debug

3 🐳 استخدام Docker

بيئة معزولة ونظيفة

# بناء Docker image
docker build -t math-competition-apk .

# تشغيل البناء
docker run -v $(pwd)/output:/output math-competition-apk
