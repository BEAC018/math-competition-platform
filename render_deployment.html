🌐 النشر الدائم على Render

نشر تطبيق المسابقات الرياضية بصفة دائمة ومجانية

التطبيق جاهز للنشر الدائم

1 🚀 إنشاء حساب Render

مجاني 100% - 750 ساعة شهرياً

🌐 اذهب إلى Render.com
📋 خطوات التسجيل:
1. اضغط "Get Started"
2. سجل بـ GitHub (الأسهل)
3. أكمل إعداد الحساب
4. مجاني تماماً للاستخدام الأساسي

2 📁 رفع الكود إلى GitHub

إنشاء مستودع جديد

📁 إنشاء Repository جديد
⚙️ إعدادات Repository:
• اسم Repository: math-competition-platform
• النوع: Public (مطلوب للخطة المجانية)
• الوصف: منصة المسابقات الرياضية
• ارفع جميع الملفات من مجلد المشروع
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/math-competition-platform.git
git push -u origin main

3 🔗 ربط Render بـ GitHub

📋 خطوات الربط:
1. في Render Dashboard، اضغط "New +"
2. اختر "Web Service"
3. اختر "Connect a repository"
4. اختر المشروع الذي رفعته
5. اضغط "Connect"

4 ⚙️ إعداد الخدمة

Name: math-competition-platform
Environment: Python 3
Build Command: ./build.sh
Start Command: gunicorn alhassan.wsgi:application
⚠️ مهم جداً:
تأكد من كتابة الأوامر بالضبط كما هو موضح أعلاه

5 🔐 متغيرات البيئة

أضف هذه المتغيرات في Environment Variables:

DEBUG=False
ALLOWED_HOSTS=.onrender.com
SECRET_KEY=your-secret-key-here
STUDENT_ACCESS_CODE=ben25
🔑 ملاحظة:
DATABASE_URL سيتم إنشاؤه تلقائياً بواسطة Render

6 🎯 النشر والانتظار

⏱️ مدة النشر:
• البناء: 3-5 دقائق
• النشر: 1-2 دقيقة
• المجموع: 5-7 دقائق تقريباً
📱 بعد النشر:
ستحتاج إنشاء مدير جديد باستخدام Shell في Render Dashboard
