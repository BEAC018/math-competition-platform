🌐 النشر السريع على Render - دليل مختصر
===========================================

⚡ خطوات سريعة (10 دقائق):
---------------------------

1. 🚀 إنشاء حساب Render:
   - اذهب إلى: https://render.com
   - اضغط "Get Started"
   - سجل بـ GitHub (الأسهل)
   - مجاني 100%

2. 📁 رفع الكود إلى GitHub:
   - اذهب إلى: https://github.com/new
   - اسم Repository: math-competition-platform
   - اجعله Public
   - ارفع جميع ملفات المشروع

3. 🔗 ربط Render بـ GitHub:
   - في Render: New + → Web Service
   - Connect a repository
   - اختر المشروع

4. ⚙️ إعدادات الخدمة:
   Name: math-competition-platform
   Environment: Python 3
   Build Command: ./build.sh
   Start Command: gunicorn alhassan.wsgi:application

5. 🔐 متغيرات البيئة:
   DEBUG=False
   ALLOWED_HOSTS=.onrender.com
   SECRET_KEY=your-secret-key-here
   STUDENT_ACCESS_CODE=ben25

6. ⏱️ انتظار النشر (5-7 دقائق)

7. 🔑 إنشاء مدير جديد:
   - في Render Dashboard → Shell
   - python manage.py createsuperuser
